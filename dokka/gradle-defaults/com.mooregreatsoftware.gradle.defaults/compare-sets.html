compareSets

fun <U : Comparable<U>> compareSets(s1: Set<U>?, s2: Set<U>?): Int

Compares two Sets of Comparable items. If one is null, that sorts lower. If ones size is smaller, that sorts lower. Otherwise the first non-equal value after they are sorted is used for the comparision.
